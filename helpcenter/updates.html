Updates Help

Unsure on how to update Harmony? Take a look below.

Updating Harmony

  1. First, tap the more_vert icon on the top bar of the main Harmony screen.

  2. Tap the settings Settings option.

  3. Then tap the system_updateUpdates option.

  4. Then tap the system_updateCheck for Updates option.

  5. Once on the GitHub releases page for Harmony, scroll down to Assets under the latest version, then tap the file ending in .apk.

  6. Once downloaded, tap on the file name in your notifications, and tap Update on the dialog that appears.
